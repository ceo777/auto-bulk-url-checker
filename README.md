# Automated Bulk URL Checker API

## Description

Automated Bulk URL Checker is based on NestJS and MongoDB. It uses perfomant [Undici](https://undici.nodejs.org) HTTP library for checking URLs asynchronously. API interface allows to add, delete and show saved URLs.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
